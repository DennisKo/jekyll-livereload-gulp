Gulp Livereload for Jekyll
-----
Quick gulp script to enable livereload for Jekyll 2.x.



- install Livereload extension
- download repo
- install 'npm install'

## Install
    $ git clone git@github.com/DennisKo/jekyll-livereload-gulp.git
    $ cd jekyll-livereload-gulp
    $ npm install

## Usage

		$ jekyll serve
		$ gulp
